# 機能要件
- ルートにアクセスしたときに勝手に初期ルートにアクセスされてしまいます
    - ルートにアクセスしたときはルームIDの入力をもとめてください
    - そののち、参加者用のURL「/room/[roomId]」にアクセスするようにしてください。
    - 画面右上にハンバーガーメニューを置いてください。
    - 中には今は「既存ルームを管理する」「ルームを作る」メニューのみ置いてください
- 管理者画面「/admin」
    - Google認証で許可したユーザーのみがアクセスできるようになります(これはCloudflare Zero Trustを用い、「62ichiken@gmail.com」「ichinose.kenki@tbs.co.jp」のみがアクセスできるようになります)
        - 認可されなかった場合、「403 アクセス権限がありません」というモーダルを表示し、ルートに戻します
    - 「既存ルームを管理する」を押下したときの画面です。
    - Googleユーザーごとにそのユーザーが管理者となったルームを保持しています。ユーザーのメールアドレスとルームIDは紐づけてSQLite(Cloudflare D1)に保存してください

- ルーム管理画面「/admin/edit?q={roomId}」
    - ルームを作成するときにも既存のルームを編集するときにもこちらの画面を開きます。
        - 初回登録時には「/admin/edit」が開きます
        - 初回登録後roomIdが保存されると「/admin/edit?q={roomId}」にリダイレクトします。また、既存のルームは「?q={roomId}」からroomIdを読み取ってそこにアクセスを行います。
    - 初回登録時のみ「保存する」を押下した際にルームの作成処理を行います。
        - 必須項目が入力されていないときは「保存する」が押下できないようにしてください。
    - 2回目以降は1秒以上編集がないことをトリガーに自動保存します。
        - 保存処理が成功したときは「ルームを保存しました」と保存ボタン上部にポップアップを出し、保存にエラーが発生したときは5回を上限に保存をリトライし、失敗したときは「保存に失敗しました。しばらくしてもう一度保存してください」とユーザーに再保存を促す。
    - bodyのヘッダー部分では
        - 「ルームID」を英数字記号のみでバリデートして自由入力させます(必須項目)
        - 「ルーム名」を自由入力させます(必須項目)
    - ルーム管理画面では、「画像の追加」「クイズの追加」「画像とクイズのドラッグアンドドロップでの並び替え」が可能です。
        - 具体的には「コンテンツ追加スペース」を作ります。GoogleSiteやnote+の編集画面のようなイメージです
            - 画像のドラッグアンドドロップでの追加を許します。
                - 画像は右クリックで削除ができます
            - クイズはスペースのダブルクリックでGoogleFormのようなユーザー入力項目を、単一選択のラジオボタンもしくは複数選択のチェックボックスでの追加として許します。
                - クイズは「問題」と「選択肢」と「解答」が必須項目であり、選択肢は2つが必須で最低、後は1つずつ増やしていけ、最高で6択までの追加が可能です。解答にかんしては解答設定コンポーネントには「選択肢」の内容をrefで反映させ、単一選択ならば1つ選択し設定、複数選択ならば複数選択肢設定することができます。
                - クイズを作成したとき、制限時間が何秒か必須項目として自由入力させます。
                - 未入力のままクイズのコンテンツスペースからフォーカスが外れた際、未入力の選択肢はフロントから消します。
            - 右クリックでも「画像の追加」と「クイズの追加」が可能です。
            - コンテンツ追加スペースでは、上から順に表示順を登録します。
            - 既に追加したコンテンツについて、ドラッグアンドドロップで順番を入れ替えることが可能です。一番左にはドラッグアンドドロップ可能なことを示すアイコンを配置してください。その左には複数選択可能なチェックボックスを配置してください。複数選択した後「選択された全てのコンテンツを削除する」から削除をすることが可能です。
    - 初回保存以降は自動保存されるが、「編集された」の判断条件は以下のよう
        - ルームID・ルーム名の変更
        - 画像の追加・更新(差し替え)・削除
        - クイズの問題・解答・制限時間の追加・更新・削除
        - 表示順の並び替え順変更
        - また、動作を軽くするため、「前回から変更のあった内容リスト」を保持し、保存時にはそのリストから登録を行い、保存後はその変更部分リストを削除して空配列にするものとする
    - また、roomIdが保存されたとき、管理者用配信画面「/admin/room/[roomId]」と参加者用画面「/room/[roomId]」へのリンクとQRコードが発行され、リンクはボタンの押下でクリップボードにコピー、QRコードは保存ボタンでそれぞれ保存ができます。
